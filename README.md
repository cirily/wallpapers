# CutefishOS Wallpaper

CutefishOS's system wallpaper.

## Build

```shell
git clone https://github.com/cirily/wallpapers.git 
cd wallpapers
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr ..
make
```

## Install

```shell
sudo make install
```
