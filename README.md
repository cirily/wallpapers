# Piscesys Wallpapers

Pisces System's wallpapers.

## Build

```shell
git clone https://github.com/piscesys/wallpapers.git 
cd wallpapers
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr ..
make
```

## Install

```shell
sudo make install
```
